# Projet 7 - Home Credit Default Risk

## Objectif
Prédire si un client remboursera son prêt.

## Structure
- `data/raw/` : données brutes
- `data/processed/` : features après preprocessing
- `notebooks/` : exploration et modélisation
- `src/` : fonctions réutilisables
- `reports/` : résultats et présentation

## Environnement
Utiliser `environment.yml` avec Conda.
